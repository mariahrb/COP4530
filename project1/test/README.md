# Travel Packing Rotation System - Tests

## How to Run

1. Navigate to the tests folder:
   ```bash
   cd GroupNumber/tests
```

2. Compile the values tests:
    ```bash
g++ -o values_tests values_tests.cpp ../services/*.cpp
./values_tests
```

Compile the errors tests:

g++ -o errors_tests errors_tests.cpp ../services/*.cpp
./errors_tests


Observe the console output for normal behavior and error handling.
